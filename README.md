# Products Store

## A full stack application with MERN.

### Tech Stack

-   **[Chakra UI](https://v2.chakra-ui.com/)** is a simple, modular and accessible component library that gives you the building blocks to build React applications.

-   **[Express.js](https://expressjs.com/)** a fast, unopinionated, minimalist web framework for Node.js

-   **[MongoDB](https://www.mongodb.com/)** a NoSQL database.

-   **[Mongoose](https://mongoosejs.com/)** Elegant MongoDB object modeling for Node.js

-   **[Node.js](https://nodejs.org/en)** an open-source, cross-platform JavaScript runtime environment.

-   **[React](https://react.dev/)** an open-source library used for mobile and web development.
